# 🔄 Автоматическая синхронизация

## ✅ Что работает автоматически

### При использовании `/remember` или `qwen-remember`:

```
1. Сохранение в PostgreSQL (mcp_memory)
2. Обновление QWEN.md
3. Git коммит
4. Push на GitHub
```

**Всё автоматически!** 🎉

---

## ⚠️ Что требует ручного действия

### Если изменения внесены не через `/remember`:

Например:
- Прямое редактирование `QWEN.md`
- Изменения в `mcp.json` или `settings.json`
- Добавление новых skills

**Тогда выполните:**
```bash
qwen-sync
# или
cd ~/.qwen && git add . && git commit -m "Update" && git push
```

---

## 📊 Сравнение

| Действие | Автосинхронизация |
|----------|-------------------|
| `/remember <факт>` | ✅ Да |
| `qwen-remember "<факт>"` | ✅ Да |
| Прямое редактирование файлов | ❌ Нет (нужен `qwen-sync`) |
| На другом компьютере (pull) | ❌ Нет (нужен `git pull`) |

---

## 🔧 На другом компьютере

После клонирования репозитория:

```bash
# 1. Клонируйте
git clone https://github.com/dimgo66/qwen-memory.git ~/.qwen

# 2. Включите автопуш (скопируйте hook)
cp ~/.qwen/.git/hooks/post-commit.example ~/.qwen/.git/hooks/post-commit 2>/dev/null || true

# 3. Теперь /remember будет синхронизироваться автоматически
```
