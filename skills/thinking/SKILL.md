# Sequential Thinking Skill

## Description
Structured problem-solving skill using sequential thinking methodology.

## Approach
1. Break down complex problems
2. Think step by step
3. Document reasoning
4. Track thought branches

## Usage
Use this skill for:
- Complex problem analysis
- Decision making with multiple factors
- Debugging intricate issues
- Planning multi-step solutions

## Benefits
- Clearer reasoning trail
- Better handling of complexity
- Easier to backtrack and revise thoughts
