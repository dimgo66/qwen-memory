# Python Skill

## Description
Python development skill for running scripts, managing environments, and packages.

## Commands
```bash
# Run scripts
python <file.py>
python -m <module>

# Virtual environments
python -m venv venv
source venv/bin/activate

# Pip
pip install <package>
pip install -r requirements.txt
pip freeze > requirements.txt
pip list
pip uninstall <package>

# Poetry
poetry install
poetry add <package>
poetry run python <file.py>

# Testing
pytest
pytest -v
pytest --cov=.
python -m unittest
```

## Usage
Use this skill for Python development, virtual environments, and package management.
