# Fullstack Development Skill

## Description
End-to-end web development skill covering frontend, backend, and database.

## Frontend Stack
- **React/Vue.js** - Modern frameworks
- **TypeScript** - Type-safe JavaScript
- **Tailwind/Bootstrap** - CSS frameworks
- **State Management** - Redux, Pinia, Vuex

## Backend Stack
- **Node.js + Express** - JavaScript backend
- **Python + FastAPI/Django** - Python backend
- **REST API** - Standard HTTP APIs
- **GraphQL** - Query language for APIs

## Database
- **PostgreSQL** - Relational database
- **MongoDB** - NoSQL document store
- **Redis** - Caching layer
- **Prisma/TypeORM** - ORMs

## DevOps
- **Docker** - Containerization
- **CI/CD** - GitHub Actions
- **Vercel/Netlify** - Frontend hosting
- **AWS/Heroku** - Backend hosting

## Usage
Use this skill for building complete web applications from database to UI.
Follow best practices for security, performance, and scalability.
