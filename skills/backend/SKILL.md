# Backend API Skill

## Description
Backend development skill for building robust APIs and services.

## REST API Design
```
GET    /resources       - List
GET    /resources/:id   - Get one
POST   /resources       - Create
PUT    /resources/:id   - Update
DELETE /resources/:id   - Delete
```

## Authentication
- **JWT** - Token-based auth
- **OAuth2** - Third-party auth
- **Session** - Cookie-based auth
- **API Keys** - Service auth

## Best Practices
- Input validation (Zod, Joi)
- Error handling middleware
- Rate limiting
- CORS configuration
- Request logging
- API documentation (Swagger/OpenAPI)

## Database Patterns
- Repository pattern
- Active Record
- Unit of Work
- CQRS

## Usage
Use this skill for designing and implementing backend APIs.
Focus on security, performance, and maintainability.
