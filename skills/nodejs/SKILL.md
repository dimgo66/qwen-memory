# Node.js Skill

## Description
Node.js development skill for running, debugging, and managing Node applications.

## Commands
```bash
# Run scripts
node <file.js>
node --inspect <file.js>

# NPM
npm install
npm install -D <package>
npm run <script>
npm test
npm run build

# Yarn
yarn install
yarn add <package>
yarn dev
yarn build

# PNPM
pnpm install
pnpm add <package>

# Process management
pm2 start <app>
pm2 list
pm2 logs
```

## Usage
Use this skill for Node.js project setup, dependency management, and running scripts.
