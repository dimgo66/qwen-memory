# UI/UX Design Skill

## Description
Frontend design skill for creating beautiful, modern user interfaces.

## Design Principles
- **Material Design** - Google's design system
- **Bootstrap** - Responsive CSS framework
- **Tailwind CSS** - Utility-first CSS
- **Color Theory** - Harmonious color palettes
- **Typography** - Readable font hierarchies
- **Spacing** - Consistent 8px grid system

## Components
```
- Buttons, Cards, Forms
- Navigation, Modals, Alerts
- Tables, Lists, Grids
- Charts, Icons, Avatars
- Responsive layouts
```

## Tools
- Figma (design mockups)
- CSS preprocessors (Sass/SCSS)
- CSS frameworks (Bootstrap, Tailwind)
- Icon libraries (Material Icons, FontAwesome)

## Usage
Use this skill for UI component design, layout planning, and styling.
Focus on accessibility, responsiveness, and visual hierarchy.
