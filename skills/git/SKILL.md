# Git Skill

## Description
Git operations skill for version control management.

## Commands
```bash
# Status
git status

# Add files
git add <file>
git add .

# Commit
git commit -m "<message>"

# Branch operations
git branch
git checkout -b <branch>
git checkout <branch>
git merge <branch>

# Remote operations
git pull
git push
git fetch

# Log
git log --oneline -10
git diff
```

## Usage
Use this skill for all Git operations including staging, committing, branching, and syncing with remote repositories.
