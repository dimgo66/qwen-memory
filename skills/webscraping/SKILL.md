# Web Scraping Skill

## Description
Web scraping and browser automation skill using Puppeteer.

## Commands
```bash
# Navigate to URL
puppeteer.navigate <url>

# Screenshot
puppeteer.screenshot

# Click element
puppeteer.click <selector>

# Type text
puppeteer.type <selector> <text>

# Evaluate JavaScript
puppeteer.evaluate <script>

# Get content
puppeteer.content
```

## Usage
Use this skill for web scraping, taking screenshots, and browser automation.
Supports headless browsing for data extraction.
