# Responsive Design Skill

## Description
Mobile-first responsive web design skill.

## Breakpoints
```css
/* Mobile First */
@media (min-width: 640px) { }  /* sm */
@media (min-width: 768px) { }  /* md */
@media (min-width: 1024px) { } /* lg */
@media (min-width: 1280px) { } /* xl */
```

## Techniques
- **Flexbox** - 1D layouts
- **Grid** - 2D layouts
- **Container Queries** - Component-based responsiveness
- **Fluid Typography** - clamp(), vw units
- **Mobile Navigation** - Hamburger menus

## Best Practices
- Mobile-first approach
- Touch-friendly targets (44px min)
- Optimized images (srcset, picture)
- Performance optimization
- Accessibility (WCAG)

## Testing
- Chrome DevTools device emulation
- Real device testing
- Lighthouse audits

## Usage
Use this skill for creating responsive layouts that work on all devices.
Test on multiple screen sizes and orientations.
