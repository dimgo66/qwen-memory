# gsd:progress

Показать текущее состояние проекта.

## Что делает

- Читает `.planning/STATE.md`
- Показывает текущую фазу и план
- Отображает прогресс
- Предлагает следующие шаги

## Как использовать

Напишите мне:
```
gsd:progress
```

## Что я покажу

```
## Текущая позиция

Фаза: [X] из [Y] ([Название])
План: [A] из [B]
Статус: [Ready / Planning / In Progress / Done]

Прогресс: [░░░░░░░░░░] X%

## Метрики

- Выполнено планов: N
- Среднее время: X мин
- Всего времени: X.X часов

## Следующие шаги

1. [Следующее действие]
2. [Альтернатива]
```

## Файлы

- `.planning/STATE.md` — текущее состояние
- `.planning/ROADMAP.md` — дорожная карта
- `.planning/PROJECT.md` — контекст проекта
