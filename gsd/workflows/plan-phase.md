# gsd:plan-phase

Детальное планирование фазы с графом зависимостей.

## Что делает

1. Читает ROADMAP.md для понимания цели фазы
2. Исследует технические детали (если нужно)
3. Создаёт детальные планы (plans) с hash-based ID
4. Определяет зависимости между планами
5. Строит граф зависимостей
6. Обновляет ROADMAP.md и STATE.md

## Как использовать

Напишите мне:
```
gsd:plan-phase 1
```
или
```
gsd:plan-phase 2
```

## Процесс

### Шаг 1: Контекст
- Читаю ROADMAP.md — цель фазы, success criteria
- Читаю STATE.md — текущее состояние
- Читаю PROJECT.md — требования

### Шаг 2: Research (если нужно)
- Исследую технологии, библиотеки, паттерны
- Проверяю совместимость
- Нахожу лучшие практики

### Шаг 3: Детальный план с Hash-based ID

**Генерация ID:**
```
План 01-01 → 01-01.a3f8
План 01-02 → 01-02.b7c1
План 01-03 → 01-03.d9e2
```

**Формат плана:**
```
План 01-01.a3f8: [название]
- Задачи: [...]
- Зависимости: [...]
- Оценка: [...]
- Статус: pending
```

### Шаг 4: Граф зависимостей

**Визуализация:**
```
01-01 (Auth)
    ├── 01-02 (API)
    │       ├── 01-04 (Frontend)
    │       └── 01-05 (Tests)
    └── 01-03 (Database)
            └── 01-04 (Frontend)
```

**Таблица зависимостей:**

| План | ID | Зависит от | Блокирует |
|------|-----|------------|-----------|
| Auth | 01-01.a3f8 | none | 01-02, 01-03 |
| API | 01-02.b7c1 | 01-01 | 01-04, 01-05 |
| Database | 01-03.d9e2 | 01-01 | 01-04 |
| Frontend | 01-04.f1a3 | 01-02, 01-03 | 01-06 |

### Шаг 5: Верификация
- Проверяю полноту планов
- Проверяю зависимости
- **Определяю критический путь**
- **Нахожу узкие места (bottlenecks)**
- Обновляю ROADMAP.md

### Шаг 6: Обновление состояния
- Обновляю STATE.md
- **Создаю dependency graph**
- Готов к выполнению

## Результат

```markdown
### Phase 1: [Name]
**Goal**: [...]
**Plans**: 3 plans
**Dependency Graph**:
```
01-01 → 01-02 → 01-04
    ↘ 01-03 ↗
```

Plans:
- [ ] 01-01.a3f8: [детальное описание]
  - Dependencies: none
  - Blocks: 01-02, 01-03
  - Estimated: 30 min
- [ ] 01-02.b7c1: [детальное описание]
  - Dependencies: 01-01
  - Blocks: 01-04, 01-05
  - Estimated: 45 min
- [ ] 01-03.d9e2: [детальное описание]
  - Dependencies: 01-01
  - Blocks: 01-04
  - Estimated: 60 min
```

## После завершения

**Следующие команды:**
- `gsd:dependency-graph 1` — посмотреть граф
- `gsd:ready` — найти готовые планы
- `gsd:execute-phase 1` — начать выполнение
